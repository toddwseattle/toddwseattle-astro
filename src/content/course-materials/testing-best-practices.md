## Overview

Testing is fundamental to building reliable software. This exercise set covers practical patterns and tools for different testing levels.

## Exercises Covered

1. **Unit Testing Fundamentals**
   - Writing testable code
   - Mocking and stubbing
   - Test organization and naming

2. **Integration Testing Patterns**
   - Testing across system boundaries
   - Database interaction testing
   - API endpoint testing

3. **End-to-End Testing**
   - User workflow automation
   - Flakiness and reliability
   - Test environment setup

## Tools and Languages

- JavaScript/TypeScript with Vitest
- Python with pytest
- Examples in multiple languages

## Difficulty Progression

Start with basic unit tests, progress through integration patterns, and finish with complex E2E scenarios.
